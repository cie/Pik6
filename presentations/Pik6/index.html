Controls

  • Use the left and right arrow keys to navigate through the slides
  • Move your Mouse to the bottom of the presentation view to reveal more controls
  • The presenter view offers a drop down menu for all slides in the presentation
  • The F5 and ESC keys hide the presentation

Programming presentations

  • An activate event is fired on the slides when they are shown
  • A deactivate event when hidden
  • A change event is fired on the document object when slides change

Simple programming example

window.addEvent('domready', function(){
    $('demoslide').addEvents({
        'activate': function(){
            this.getElements('h2').set('morph', {
                duration: 1000
            }).morph({
                'color': ['#FFF', '#AE1215'],
                'font-size': ['50%', '100%']
            });
        }
    });
});

Scripting without scripts

  • The presentation is just an iframe with a url containing a hash
  • The slide with the id that matches the hash is made visible using the :target selector from CSS3
  • This principle allows you to trigger actions without using JavaScript and create slide transitions!

Simple CSS "scripting" example

h2#special {
    color: #FFF;
    font-size: 50%;
}
:target h2#special {
    transition: all 1s;
    color: #AE1215;
    font-size: 100%;
}
