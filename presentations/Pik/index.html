An introduction to PIK6

HTML5 presentation framework

Press → to learn more

Features

  • 100% HTML5, CSS and JavaScript (MooTools)
  • Multiple presentation views and presenter mode
  • Programmable slides
  • As fancy or a simple as you want it to be

Why use it?

  • Presenter mode
  • Hackable, scriptable presentations
  • Easy embedding of web pages, media, youtube videos â€¦
  • Great performance in modern browsers even with hundreds of slides

Why not?

  • Requires a modern web browser
  • Requires a web server to run from
  • Shares all drawbacks common to HTML slides (no GUI to create presentations, no PDF export, images hard to use)

Getting started

  1. Download Pik6 from Github or git clone git://github.com/SirPepe/Pik6.git
  2. Place the files in a directory on your web server
  3. Open Pik6 in a browser, click the "Browse" link to navigate through the slide sets

Controls

  • Use the left and right arrow keys to navigate through the slides
  • Move your Mouse to the bottom of the presentation view to reveal a panel with more controls
  • The F5 and ESC keys hide the presentation

Presenter mode

  • Open presenter.html in a new window to use the presenter mode.
  • The presenter mode offers more control elements, displays a timer and lets you see the currently displayed slide and which slide comes next
  • Requires a browser that supports Shared Workers (Chrome 8+ or Opera 11+)

Building presentations

<!-- /presentations/Whatever/whatever.html -->
<script src="../../lib/pik.js"></script>
<div id="Pik-hide"></div>

<div id="Pik-presentation-container">
    <div class="pik-slide">First Slide</div>
    <div class="pik-slide">Second Slide</div>
</div>

You can use presentations/Template/index.html as a starting point.

Styling presentations

  • Styles (simple CSS files) reside in the styles folder
  • Styles includes in PIK6:
    • Default (used in this presentation)
    • Dark (simple white text on a dark backgroud)
    • Fancy (demonstrates slide transitions, to be used with Default)
  • Use styles/Template/style.css as a starting point for your own style

Slide transitions

  • Slide visibility is completly managed by css; all slides except the one with the class pik-current are invisible by default
  • By changing the behavior of this class and the classes pik-next and pik-prev it is possible to create slide transitions
  • Try it! Add a stylesheet with transitions (remove here)

Programming presentations

  • An slideactivate event is fired on the slides when they are shown
  • A slidedeactivate event when hidden
  • A slidechange event is fired on the window object when slides change

Simple programming example

window.addEvent('domready', function(){
    $('Demo').addEvents({
        'slideactivate': function(){
            do_something();
        },
        'slidedeactivate': function(){
            do_something_else();
        }
    });
});
